# Recency Rules

## General Freshness

- Prefer currently live issuer pages.
- Lower confidence when a source is stale, cached, or undated.

## `/card-news`

- Use a 3-month lookback window only.
- Include direct card changes, issuer updates that materially affect the card, and major approved-site coverage that changes how the card should be understood.
- Exclude generic issuer chatter or evergreen “best card” articles that do not describe a recent change.

## Other Commands

- Current public offers should be treated as time-sensitive.
- Fees, credits, earning rates, and transfer partners should be rechecked against live sources when the command is used.
