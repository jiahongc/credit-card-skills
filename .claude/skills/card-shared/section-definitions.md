# Section Definitions

## `/card-full` Composition

`/card-full` is the top-level report. It reuses the same conceptual sections exposed by focused commands and adds the broader analytical categories from the framework. Each section uses a compact block with one emoji heading and condensed facts.

### Sections

- `💰 Fees`
  - annual fee, authorized user fee, foreign transaction fee, APR caveat, balance transfer fee, cash advance fee, late fee
- `🎁 Welcome Offer`
  - public bonus, spend requirement, qualification window, eligibility restrictions, lifetime/family language
- `📈 Earning Rates`
  - reused by `/card-rate`
- `🔄 Redemption`
  - transfer program summary and partners reused by `/card-transfer`
  - portal/cash-out/minimum redemption caveats
- `🏷️ Credits`
  - reused by `/card-credits`
- `✈️ Travel Benefits`
  - lounge access, hotel benefits, rental car benefits, travel protection benefits, travel credits
- `🛡️ Protections`
  - purchase protection, extended warranty, return protection, cell phone protection, fraud protections
- `⚙️ Account Mechanics`
  - virtual cards, authorized user handling, app/alert capabilities, dispute/autopay notes
- `✅ Eligibility`
  - issuer family rules and known restriction language
- `🧭 Strategy`
  - downgrade paths, no-fee fallback, ecosystem role, keeper value after year one
