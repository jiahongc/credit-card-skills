# Example Prompts

## Direct Invocation

```text
/credit-card-info Chase Sapphire Preferred
```

## With Clarification Request

```text
/credit-card-info Hilton Honors American Express Surpass Card
Focus on hotel perks and statement credits, but still return the full dossier format.
```

## Expected Behavior Notes

- The skill should identify the exact card variant before listing benefits.
- The skill should not compare products or recommend alternatives.
- The skill should explicitly note conflicts between issuer and secondary sources.
